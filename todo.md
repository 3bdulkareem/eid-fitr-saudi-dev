## ملاحظات المشروع

### التقنيات المستخدمة
- **Frontend**: React 19 + TypeScript + Tailwind CSS 4
- **Backend**: Express + tRPC + Node.js
- **Database**: MySQL + Drizzle ORM
- **Authentication**: Manus OAuth
- **UI Components**: shadcn/ui + Lucide Icons
- **State Management**: React Hooks + TanStack Query
- **Styling**: Tailwind CSS with custom theme
- **Testing**: Vitest (للاختبارات المستقبلية)

### الملفات الرئيسية
- `client/src/pages/Home.tsx` - الصفحة الرئيسية مع جميع الميزات
- `client/src/App.tsx` - التوجيه الرئيسي
- `server/routers.ts` - إجراءات tRPC
- `drizzle/schema.ts` - تعريف قاعدة البيانات
- `client/src/index.css` - الأنماط العامة
- `todo.md` - هذا الملف
